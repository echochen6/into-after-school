# INTO · After School

面向中国小学生的**课后晚托**平台原型:机构负责接送、包餐、监督写作业(不含学科辅导),家长在手机上实时看到孩子的每一步,并通过取件码安全交接。含美国远程英语课(合规增值,规划中)。

> 当前为**纯前端 Demo**(假数据、浏览器本地存储),用于验证产品形态与体验,尚未接后端。

## 怎么打开

**不需要安装任何东西**——直接双击 `index.html`,在浏览器里打开即可(离线也能用)。

- 顶部开关切换 **家长视图 / 老师视图**
- 右上角 **「商业蓝图」** = 投资人视角的路演页
- 右上角 **「重置」** = 恢复初始演示数据

改动代码后,在浏览器里按 `Cmd + Shift + R`(Mac)强制刷新即可看到最新版。

## 主要功能(Demo)

- 今日时间线:接到 → 用餐 → 写作业 → 可接走 → 已交接,实时打时间戳
- 安全交接:授权接送人 + 动态取件码,老师核对后交接
- 作业完成清单、今日菜单、今日照片、每日小结卡(可分享)
- 英语课排课(外部视频链接占位)
